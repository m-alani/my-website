Healthwise Clinic

Web Development Responsive HTML5 CSS3 Bootstrap JavaScript PHP jQuery SEO

The Healthwise Clinic website was more of a weekend challenge rather than a fully planned project. The challenge was to design and code a responsive website over a weekend from scratch. I gladly took on the challenge, and this is what I came up with.

For the most part, healthwiseclinic.ca is a simple website with heavy focus on front-end. And for that (the front-end), I decided to go with Bootstrap for the main design, since my friend needed the website to be responsive. I kept the tinkering in jQuery to a minimum since I had a time limit, so I just used the default bootstrap arsenal. I wrote some simple JavaScript validation to reduce the stress on my main validation code (written in PHP on the back-end) for requesting appointments. I have also put in some simple SEO optimizations (link to Google+, Facebook description, etc.) just to make things easier for the business owner.

Overall, it was a fun challenge which I enjoyed. I wouldn't change much if I didn't have the time limit, except for cleaning up the markup to make it more W3C compliant.

Visit healthwiseclinic.ca

MedicSpa was an interesting project I worked on for about 2 weeks. Osgoode Mobile

iOS Swift XCode Sketch Ubuntu Server Node.js Parse Server CocoaPods MongoDB Agile Apple's App Store MarkDown Technical Documentation Lead Developer

As the Lead Developer for this project, I was in charge of everything regarding the app. I did the initial designs and requirement gathering, I wrote the technical plan, I planned both the front and back ends, and presonally coded everything on the frontend. I was aiming to implement as many Agile concepts as possible into the project, and given the small team size, we ended up tracking sprints on Trello, and doing daily standing SCRUMs. We also used a private GitLab instance (which I personally built, on a previous project) for version control.

The frontend was written in Swift on XCode, and used various APIs and SDKs: CoreData, GCD, UIKit, Parse iOS SDK, and few more.

The rest of the team worked on coding the backend which I designed. I participated by writing few custom Node.js modules. The backend is built on a Ubuntu Server, running Parse Server with an Apache2 instance as a reverse-proxy, with few custom Node.js modules, and MongoDB.
